## Digi-Cloq
		
## Overview

	* Digi-Cloq is a digital clock apprication.
			
## Description

	* Digi-Cloq is using openGL. Its Interface is based on glut, and written in C++.
	* Makefile is available for Linux. 
	* As for windows and MacOS,  modifying the Makefile.
		  
		
## Requirements

	* freeglut
